JavaScript’dan kuchliroq va ishonchliroq dasturlash tili

1-dars TypeScript nima?

1️⃣ TypeScript nima?

TypeScript — bu Microsoft tomonidan yaratilgan va JavaScript’ning kengaytirilgan versiyasi. U kuchli tiplar (strong types), interfeyslar va boshqa qo‘shimcha imkoniyatlar bilan JavaScript’ni yanada qulay qiladi.

2️⃣ O‘zgaruvchilar

TypeScript’da o‘zgaruvchi e’lon qilishda let yoki const ishlatiladi va uning turi aniqlanadi:

let age: number = 25;
let name: string = "Ali";
const isActive: boolean = true;

3️⃣ Funksiyalar

Funksiyalarda ham argument va qaytish turi ko‘rsatiladi:

function add(a: number, b: number): number {
    return a + b;
}
